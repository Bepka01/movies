# movies

Проект переведен на сборку Vite с поддержкой SCSS.

## Команды

- `npm install` - установка зависимостей
- `npm run dev` - локальная разработка
- `npm run build` - production-сборка
- `npm run preview` - просмотр собранного `dist`

## Стили

- Исходники стилей только в `src/scss/`
- Главная страница: `src/scss/main.scss`
- Авторизация: `src/scss/auth.scss`
- Общие переменные: `src/scss/variables.scss`
- Общий reset: `src/scss/reset.scss`
